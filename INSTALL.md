## Installation

### Requirements
- Python 3.12
- Pytorch
- CUDA 12.X

### Example conda environment setup
```bash

conda create --name sarwmix python=3.12 -y
conda activate sarwmix

pip3 install torch torchvision

pip install -r requirement.txt
```
